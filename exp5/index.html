📚 Features & Implementation Details

Quiz Application Features:

A fully functional quiz application with the following capabilities:

  • Displays 5 MCQ questions with 4 options each
  • setInterval(): Runs countdown timer every 1000ms
  • clearInterval(): Stops timer when time reaches zero or quiz is submitted
  • Shows "Only 5 seconds remaining!" warning message when timer hits 5 seconds
  • Automatically submits quiz when timer reaches 0
  • Disables all input options after submission using the disabled property
  • Calculates and displays final score out of 5
  • Shows detailed results with correct/incorrect indicators
  • Allows quiz restart for multiple attempts

Event Countdown Timer Features:

A flexible countdown timer for event registration with these capabilities:

  • Set custom future date and time using datetime input
  • Date Object: Calculates time difference using getTime()
  • Mathematical Conversion: Converts milliseconds to Days, Hours, Minutes, Seconds
  • setInterval(): Updates display every 1000ms (every second)
  • clearInterval(): Stops timer when event time is reached
  • Displays "Registration Closed" message when countdown reaches zero
  • Real-time progress bar showing percentage to event
  • Dynamic status messages based on time remaining
  • Zero-padded number format (e.g., "05" instead of "5")
  • Reset functionality to set new event dates

Key JavaScript Concepts Used:

Timer Management:
